1. Команда cd pmat/. Открываем нужную папку
2. Команда mkdir lab1/. Создаём папку для лабораторной работы №1
3. Команда touch rand.py touch division.py touch sqrt.py. Создаём файлы
4. Команда vim rand.py vim division.py vim sqrt.py. Пишем код в файлах
5. Команда chmod +x. Делаем файлы исполняемыми
6. Команда ./rand.py./division.py./sqrt.py>output.txt.Запускаем конвейер
7. Результат сохранён в файл output.txt
