1. Команда cd pmat/. Открываем нужную папку
2. Команда mkdir lab2/. Создаём папку для лабораторной работы №1
3. Команда touch greeter.py. Создаём файл
4. Команда vim greeter.py. Пишем код в файл
5. Команда chmod +x greeter.py. Делаем файлы исполняемыми
6. Команда ./greeter.py <names.txt.Запускаем 1 способом
7. Команда ./greeter.py.Запускаем 2 способом (заканчиваем ввод командой CTRL C)
